# Support

FastADB is a community-supported beta project.

## Questions

Use GitHub Discussions if enabled. Otherwise open a question issue and include:

- OS and version.
- FastADB version.
- ADB version.
- scrcpy version, when relevant.
- A short description of what you are trying to do.

## Bugs

Use the bug report template. Include reproduction steps and logs when possible.

## Security

Follow `SECURITY.md`. Do not disclose vulnerabilities in public issues.
